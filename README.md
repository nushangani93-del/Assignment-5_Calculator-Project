# Assignment-5_Calculator-Project
The Basic Calculator is a simple interactive application developed using Python.
Project Description:

This project is a console-based Basic Calculator developed using the Python programming language. The main objective of this project is to create a simple mathematical calculator that performs fundamental arithmetic operations through a menu-driven interface.

The calculator allows the user to select different mathematical operations such as:

Addition

Subtraction

Multiplication

Division

Second Power (Square)

Square Root

Exit

The program continuously displays a menu and allows the user to choose an operation repeatedly until the user selects the exit option. This ensures that the calculator remains active and user-friendly.

The program accepts numerical inputs from the user and performs the selected operation. It also includes error handling to manage invalid inputs, division by zero, and square root of negative numbers. This improves the reliability and robustness of the application.

Objectives of the Project:

To understand basic Python programming concepts.

To implement conditional statements (if-elif-else).

To use loops (while loop) for repeated execution.

To apply mathematical operations in Python.

To handle runtime errors properly.

To build a user-interactive console application.

Tools and Technologies Used:

Programming Language: Python

Environment: Jupyter Notebook / Python IDE

Libraries Used: math (for square root function)

Key Features:

Menu-driven program

Repeats until user chooses Exit

Handles invalid menu choices

Prevents division by zero

Prevents square root of negative numbers

Simple and easy-to-use interface

Conclusion:

The Basic Calculator project demonstrates fundamental programming skills in Python, including loops, functions, condition handling, and error management. It serves as a beginner-friendly application that helps understand how user input and logical operations work together in a real-world program.
